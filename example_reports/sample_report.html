Quantification

Although all samples have comparable number of sequenced reads (around 15 million), the number decreases at each step of the alignment process:

  1. Sequenced reads
  2. Sequenced reads that align to the genome
  3. Sequenced reads that align to the genome in a single position
  4. Sequenced reads that align to the genome in a single position that belongs to a gene

We shouldnt use the 08 dataset, it doesnt seem to have enough sequenced reads.

We cant just use the number of reads that align to each gene (raw counts) as an indicator of expression because

  • the sequencing depth (or coverage) is different for each sample, so we normalize each count to account for the sequencing depth in each sample.
  • the RNA composition (viral vs. host, or highly-expressed vs lowly-expressed genes) is different in each sample, so we calculate a set of scaling factors that minimize the log-fold changes between the samples for most genes.

Even though at 6hpi and 18hpi there are less reads that map uniquely to a gene than at other timepoints (due to the viral infection), the distribution of counts is similar for all samples (notice there are many more genes with low counts at 18hpi).
